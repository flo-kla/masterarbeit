# Supplementary Materials

This repository contains the supplementary materials for the master’s thesis:

**“Konvergenz ethischer Werte in KI-Richtlinien der DAX-Unternehmen: Eine vergleichende Inhaltsanalyse und Governance-Bewertung”**

The repository is intended to support transparency, traceability, and reproducibility of the research process.

## Repository Structure

### `CODEBOOK`
Contains the coding framework used for the content analysis, including category definitions, coding rules, and the audit trail documenting relevant coding decisions, refinements, and revisions during the research process.

### `ANHANG_ERGEBNISTABELLEN`
Contains the result tables and supplementary overviews generated during the analysis. These materials complement the results section of the thesis and serve as appendix-related documentation.

### `INCLUDED_DOCUMENTS`
Contains the corporate documents included in the final sample and used for coding and analysis.

### `LITERATURQUELLEN`
Contains the key literature sources used for the theoretical foundation, methodological framing, and contextualization of the study.

## Purpose

The materials provided in this repository document the empirical basis of the thesis and make the analytical procedure nachvollziehbar. They are shared to enhance methodological transparency and to facilitate academic review of the coding and evaluation process.

## Note

This repository includes only materials that were relevant for the final analysis and could be documented in a structured and transparent manner.
